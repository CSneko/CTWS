CSNKTools

How To Use

  1. Download the latest version of the CSNKTools plugin from Github

  2. Put the downloaded plugins in the 'plugins' folder

  3. Restart the server and wait for the plugin to load

  4. If an error occurs, restart it again

Configuration

Main Configuration(config.yml)


# Plugin master switch
Enable: true
#Set language (zh-cn/en-eu)
language: zh-cn
player:
  join:
   # Whether to open the login message
   Enable: true
   # Message sent when logging in
   Message: Welcome %player_name% to join the server
Scoreboard:
  # Whether to enable the scoreboard
  Enable: true
  # Scoreboard title
  title: CSNKTools
  # Scoreboard content
  line:
   - 名称: %player_name%
   - 等级: %player_level%
   - 延迟: %player_ping%
  # Scoreboard update time (tick)
  update: 20
